Configuration

Ajoutez votre profil de correspondances personnalisé

Pour ajouter votre propre profil de correspondances, ajoutez le au fichier module.config.php comme ceci:

'marcxmlexport_mapping' => [
    'factories' => [
        'your_mapping' => Path\To\Your\Mapping::class,
    ],
];

Nettoyage récurrent par cronjob

Note

Nécessite un script issu d’un repository public de BibLibre.

Un script de nettoyage peut être lancé pour supprimé les fichiers stockés dans ~/files/Marc_XML_Export/. Il prend un argument date qui représente la date limite à partir de laquelle les exports doivent être conservés.

~/tools/omeka-s/job-start --user-id X --job-class 'MarcXmlExport\JobRemoveFilesJob' --job-args '{"date": "<YEAR>-<MONTH>-<DAY>"}'
